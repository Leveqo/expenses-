# expenses-

Это небольшая консольная программа для учета личных расходов.
В ней можно будет добавлять категории, записывать траты и смотреть общую сумму расходов.

Я делаю этот проект на Python как учебную работу.

Чтобы запустить программу, нужно выполнить:

```bash
python expenses.py
```

В программе можно:

- добавить категорию;
- добавить расход;
- посмотреть список расходов;
- посмотреть общую сумму расходов;
- посмотреть расходы или сумму по одной категории.

Примеры команд:

```bash
python expenses.py add-category еда
python expenses.py add 250 еда обед
python expenses.py list
python expenses.py list еда
python expenses.py total
python expenses.py total еда
```

Данные сохраняются в файл `expenses.json`.
